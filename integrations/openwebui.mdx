# Using OpenWebUI with NanoGPT's API

Learn how to set up OpenWebUI to work with NanoGPT's API and access all available models.

## Quick Setup Guide

1. Get your API key from [nano-gpt.com/api](https://nano-gpt.com/api)
2. In OpenWebUI, go to Settings and find the "Add Connection" option
3. Enter the following details:
   - URL: `https://nano-gpt.com/api/v1`
   - API Key: Your key from step 1
   - Prefix ID: (Optional) Leave empty or add a custom prefix
   - Model IDs: Leave empty to include all available models

That's it! Once you save the connection, you'll have access to all NanoGPT models through OpenWebUI.

## Advanced Features

### RAG (Retrieval-Augmented Generation)

OpenWebUI supports RAG capabilities, allowing you to:

1. Upload documents (PDF, TXT, etc.) to provide context
2. Have the AI answer questions based on your documents
3. Set up a knowledge base from your documents

To use this feature:
- Go to the "RAG" section in settings
- Upload your documents
- Select "Use RAG" when creating a new chat

### Prompt Templates

Create reusable prompt templates:

1. Navigate to "Prompt Templates" in settings
2. Click "New Template"
3. Define your template with variables (e.g., `{{input}}`)
4. Save the template
5. Access it from the chat interface

### Multi-Modal Support

For models that support images:

1. Click the attachment icon in the chat input
2. Upload an image
3. Ask questions about the image
