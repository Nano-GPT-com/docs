## Setting a Default Model

To set a default model, add an `agents` section to your config with the desired model as the primary:

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "nanogpt/anthropic/claude-opus-4.5"
      }
    }
  }
}
```

You can add more models by following the same format in the `models` array. For a complete list of available models and their pricing, visit our [pricing page](https://nano-gpt.com/pricing).
