# Chrome Extension

The [NanoGPT Chrome Extension](https://chromewebstore.google.com/detail/nanogpt-search-extension/ajikfdfilhfkakicgdnjhklpfcohibfg?pli=1) is now available! Query NanoGPT from any website by hitting ALT + S (Windows) or CMD + S (Mac)! See also [this short Youtube video](https://www.youtube.com/watch?v=gndGYqvnQTY).
